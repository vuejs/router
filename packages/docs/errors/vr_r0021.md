# VR_R0021: No active route record on reactivation

> No active route record was found when reactivating component with navigation guard. This is likely a bug in vue-router. Please report it.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Report with a minimal reproduction at
<https://github.com/vuejs/router/issues/new/choose>.
