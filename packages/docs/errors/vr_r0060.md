# VR_R0060: `<router-view>` used directly inside another component

> &lt;router-view&gt; can no longer be used directly inside &lt;{comp}&gt;. Use slot props instead.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Use the `<router-view>` slot to wrap the resolved component in the
transition/keep-alive:

```vue-html
<router-view v-slot="{ Component }">
  <transition>
    <component :is="Component" />
  </transition>
</router-view>
```
