# VR_R0029: Component uses `defineAsyncComponent()`

> Component "{name}" in record with path "{path}" is defined using "defineAsyncComponent()". Write "() => import('./MyPage.vue')" instead of "defineAsyncComponent(() => import('./MyPage.vue'))".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Pass `() => import("./MyPage.vue")` directly instead of wrapping it in
`defineAsyncComponent()`.
