# VR_R0020: Guard composable called outside `<router-view>`

> No active route record was found when calling `{fn}()`. Make sure you call this function inside a component child of &lt;router-view&gt;. Maybe you called it inside of App.vue?

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Call the composable from a page component or a component child of a page
component.
