# VR_B0006: Unrecognized query param default

> Unrecognized default value in definePage() for query param "{paramName}". Typeof value: "{type}".

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

This is a bug or a missing value type. Open an issue at
<https://github.com/vuejs/router> and provide the `definePage()` code.
