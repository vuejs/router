# VR_R0050: Invalid `to` prop in `useLink()`

> Invalid value for prop "to" in useLink()

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Pass a valid route location (a string path or an object) to the `to` prop.
