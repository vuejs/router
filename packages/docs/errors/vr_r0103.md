# VR_R0103: Named route with an unnamed empty-path child

> The route named "{name}" has a child without a name, an empty path, and no children. This is probably a mistake.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Move the name to the empty-path child, or give the child its own name.
