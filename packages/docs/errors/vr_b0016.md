# VR_B0016: Error decoding a value

> Error decoding "{text}". Using original value

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Ensure the value is correctly percent-encoded.
