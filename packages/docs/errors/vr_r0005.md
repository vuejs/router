# VR_R0005: Invalid location passed to `router.resolve()`

> router.resolve() was passed an invalid location. This will fail in production.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Pass a valid route location: a string path or an object with `path` or `name`.
