# VR_B0007: Route `alias` must be a string literal or array

> route alias must be a string literal or an array of string literals. Found in "{id}".

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Use a string literal or an array of string literals for the route `alias`.
