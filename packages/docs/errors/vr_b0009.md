# VR_B0009: Invalid extension

> Invalid extension "{ext}". Extensions must start with a dot.

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Prefix the extension with a dot, e.g. `.vue`.
