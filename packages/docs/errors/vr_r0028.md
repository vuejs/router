# VR_R0028: Component is a Promise instead of a function

> Component "{name}" in record with path "{path}" is a Promise instead of a function that returns a Promise. Did you write "import('./MyPage.vue')" instead of "() => import('./MyPage.vue')" ? This will break in production if not fixed.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Wrap the import in a function: `() => import("./MyPage.vue")`.
