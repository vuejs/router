# VR_R0011: Unexpected error when starting the router

> Unexpected error when starting the router:

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Inspect the actual cause; a navigation guard or async component likely threw
during the initial navigation.
