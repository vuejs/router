# VR_R0120: Error with `push`/`replace` State

> Error with push/replace State

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

The browser rejected the History API call; check for cross-origin or rate-limit
issues.
