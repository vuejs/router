# VR_B0017: Unnamed parameter in a route

> A parameter without a name was found in the route "{fullPath}" in segment "{path}".

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

This is a bug. Please report it at <https://github.com/vuejs/router>.
