# VR_R0100: Discarded invalid params

> Discarded invalid param(s) "{params}" when navigating.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Only pass params that exist on the target route. See the
[4.1.4 changelog](https://github.com/vuejs/router/blob/main/packages/router/CHANGELOG.md#414-2022-08-22)
for details.
