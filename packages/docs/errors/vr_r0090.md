# VR_R0090: Duplicated param names

> Found duplicated params with name "{name}" for path "{path}". Only the last one will be available on "$route.params".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Give each param a unique name within the path.
