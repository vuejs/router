# VR_B0018: Cannot determine if a parser is raw

> Cannot statically determine if "parser" is raw in "{filename}" because it is re-exported from "{source}". The generated route param types may be incorrect.

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Define the parser inline with `defineParamParser`/`defineParamParserRaw` instead
of re-exporting it.
