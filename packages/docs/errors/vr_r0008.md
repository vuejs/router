# VR_R0008: Invalid redirect

> Invalid redirect found:
> {target}
> when navigating to "{to}".
> A redirect must contain a name or path. This will break in production.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Return a location with a `name` or `path` from the redirect.
