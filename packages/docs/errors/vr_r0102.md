# VR_R0102: Alias param mismatch

> Alias "{alias}" and the original record: "{original}" must have the exact same param named "{name}"

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Use the same param names in the alias as in the original route.
