# VR_R0022: Guard composable called outside `setup()`

> {fn}() was called outside of component setup but it must be called at the top of a setup function

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Call it synchronously at the top of `setup()`, before any `await`.
