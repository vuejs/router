# VR_R0007: Hash missing the leading `#`

> A `hash` should always start with the character "#". Replace "{hash}" with "#{hash}".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Prefix the hash value with `#`.
