# VR_R0030: Component function does not return a Promise

> Component "{name}" in record with path "{path}" is a function that does not return a Promise. If you were passing a functional component, make sure to add a "displayName" to the component. This will break in production if not fixed.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Return a Promise from the function, or add a `displayName` to the functional
component.
