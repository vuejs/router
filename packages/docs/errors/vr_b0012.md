# VR_B0012: Invalid JSON5 in a route block

> Invalid JSON5 format of `<{type}>` content in {filePath}

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Fix the JSON5 syntax in the route custom block.
