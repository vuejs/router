# VR_B0010: Segment missing the closing `)`

> Segment "{segment}" is missing the closing ")". It will be treated as a static segment.

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Add the closing `)` to the dynamic segment.
