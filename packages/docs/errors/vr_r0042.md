# VR_R0042: Element not found by `scrollBehavior` selector

> Couldn't find element using selector "{el}" returned by scrollBehavior.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Return a selector that matches an existing element, or guard against missing
elements.
