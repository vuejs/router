# VR_B0002: `definePage()` references local bindings

> \[{id}]: {message}

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Avoid referencing `<script setup>` bindings inside `definePage()`; pass static
values instead.
