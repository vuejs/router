# VR_R1003: `commit()` called with no staged data

> Loader "{key}"'s "commit()" was called but there is no staged data.

**Level:** Warning

Runtime warning from the experimental [Data Loaders](/data-loaders/), logged with `console.warn` during development and stripped from production builds.

## How to fix it

Ensure the loader resolved before calling `commit()`.
