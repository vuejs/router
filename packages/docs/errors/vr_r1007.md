# VR_R1007: Data Loaders set up twice

> Data Loader was setup twice. Make sure to setup only once.

**Level:** Warning

Runtime warning from the experimental [Data Loaders](/data-loaders/), logged with `console.warn` during development and stripped from production builds.

## How to fix it

Call the data loader plugin setup only once when creating the app.
