# VR_R0041: Invalid selector

> The selector "{el}" is invalid. If you are using an id selector, make sure to escape it.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Escape the selector or use `CSS.escape()` before passing it. See
[escaping characters in selectors](https://mathiasbynens.be/notes/css-escapes)
or [`CSS.escape`](https://developer.mozilla.org/en-US/docs/Web/API/CSS/escape).
