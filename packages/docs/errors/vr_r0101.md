# VR_R0101: Matcher received a relative path

> The Matcher cannot resolve relative paths but received "{path}".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Pass an absolute path (starting with `/`) to the matcher. Unless you called
`matcher.resolve()` directly, this is probably a bug, please open an issue at
<https://github.com/vuejs/router/issues/new/choose>.
