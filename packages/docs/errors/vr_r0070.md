# VR_R0070: Relative location without an absolute path

> Cannot resolve a relative location without an absolute path. Trying to resolve "{to}" from "{from}". It should look like "/{from}".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Resolve relative locations from an absolute path (starting with `/`).
