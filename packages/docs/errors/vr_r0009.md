# VR_R0009: Possible infinite redirection

> Detected a possibly infinite redirection in a navigation guard when going from "{from}" to "{to}". Aborting to avoid a Stack Overflow.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Only return a new location from a guard when you actually need to redirect;
otherwise return nothing.
