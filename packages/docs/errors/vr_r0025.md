# VR_R0025: `next()` callback is deprecated

> The `next()` callback in navigation guards is deprecated. Return the value instead of calling `next(value)`.

**Level:** Deprecation

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Return the value (or a location) from the guard instead of calling `next()`.
