# VR_R1006: `useQuery()` key collides with a loader

> A query was defined with the same key as the loader "[{key}]". If the "key" is meant to be the same, you should directly use the data loader instead. If not, change the key of the "useQuery()".

**Level:** Warning

Runtime warning from the experimental [Data Loaders](/data-loaders/), logged with `console.warn` during development and stripped from production builds.

## How to fix it

Use a distinct key for `useQuery()`, or use the data loader directly.
