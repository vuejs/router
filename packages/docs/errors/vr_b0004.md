# VR_B0004: Route `name` must be a string literal

> route name must be a string literal or false. Found in "{id}".

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Use a string literal or `false` for the route `name`.
