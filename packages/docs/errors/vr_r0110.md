# VR_R0110: Hash base must end with `#`

> A hash base must end with a "#": "{base}" should be "{suggestion}".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

End the hash base with a `#`.
