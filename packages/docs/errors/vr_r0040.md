# VR_R0040: Selector should be passed as an element

> The selector "{el}" should be passed as "el: document.querySelector('{el}')" because it starts with "#".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Pass the resolved element via `document.querySelector()` instead of the raw
selector string.
