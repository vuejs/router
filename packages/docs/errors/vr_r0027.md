# VR_R0027: Invalid component in a record

> Component "{name}" in record with path "{path}" is not a valid component. Received "{received}".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Pass a component or a function returning a Promise that resolves to one.
