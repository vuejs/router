# VR_B0015: Unsupported route block language

> Language "{lang}" for `<{type}>` is not supported. Supported languages are: json5, json, yaml, yml. Found in {filePath}.

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Use one of the supported languages: `json5`, `json`, `yaml`, `yml`.
