# VR_R1004: Unregistered loader returned a `NavigationResult`

> A loader returned a NavigationResult but is not registered on the route. Did you forget to "export" it from the page component?

**Level:** Warning

Runtime warning from the experimental [Data Loaders](/data-loaders/), logged with `console.warn` during development and stripped from production builds.

## How to fix it

Export the loader from the page component so it is registered on the route.
