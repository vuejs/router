# VR_B0019: Param parser not found

> Parameter parser "{parser}" not found for route "{fullPath}".

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Define the param parser or use one of the native parsers.
