# VR_R1008: Data Loaders are experimental

> Data Loader is experimental and subject to breaking changes in the future.

**Level:** Warning

Runtime warning from the experimental [Data Loaders](/data-loaders/), logged with `console.warn` during development and stripped from production builds.

## How to fix it

Pin the vue-router version if you rely on the experimental Data Loaders API.
