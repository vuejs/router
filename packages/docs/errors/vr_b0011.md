# VR_B0011: Empty parameter name

> Invalid parameter in path "{segment}": parameter name cannot be empty. Using default name "pathMatch" for ':()'.

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Give the parameter a name, e.g. `:id`.
