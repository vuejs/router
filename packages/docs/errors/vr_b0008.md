# VR_B0008: Alias array must contain only string literals

> route alias array must only contain string literals. Found {found} in "{id}".

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Only use string literals inside the route `alias` array.
