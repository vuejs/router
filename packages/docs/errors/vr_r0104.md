# VR_R0104: Absolute child path param mismatch

> Absolute path "{path}" must have the exact same param named "{name}" as its parent "{parent}".

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Include the parent route params in the absolute child path.
