# VR_R0006: Params ignored on a path location

> Path "{path}" was passed with params but they will be ignored. Use a named route alongside params instead.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Use a named route `{ name, params }` instead of `{ path, params }`.
