# VR_R1001: Loader has a different parent

> ❌👶 "{key}" has a different parent than the current context. This shouldn't be happening.

**Level:** Warning

Runtime warning from the experimental [Data Loaders](/data-loaders/), logged with `console.warn` during development and stripped from production builds.

## How to fix it

This is a bug. Please report a reproduction at <https://github.com/vuejs/router/>.
