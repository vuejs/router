# VR_R1009: Returning a `NavigationResult` from a loader is deprecated

> Returning a NavigationResult from a loader is deprecated. Use reroute() instead, which throws internally.

**Level:** Deprecation

Runtime warning from the experimental [Data Loaders](/data-loaders/), logged with `console.warn` during development and stripped from production builds.

## How to fix it

Call `reroute()` instead of returning a `NavigationResult`.
