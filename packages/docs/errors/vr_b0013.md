# VR_B0013: Invalid JSON in a route block

> Invalid JSON format of `<{type}>` content in {filePath}

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Fix the JSON syntax in the route custom block.
