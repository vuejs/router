# VR_R0003: Resolved location starts with multiple slashes

> Location "{location}" resolved to "{href}". A resolved location cannot start with multiple slashes.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Remove the leading slashes from the location or fix the route configuration.
