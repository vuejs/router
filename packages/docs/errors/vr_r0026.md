# VR_R0026: Record missing `component(s)` or `children`

> Record with path "{path}" is either missing a "component(s)" or "children" property.

**Level:** Warning

Runtime warning logged with `console.warn` during development and stripped from production builds.

## How to fix it

Add a `component`, `components`, or `children` to the route record.
