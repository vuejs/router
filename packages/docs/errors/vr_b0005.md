# VR_B0005: Route `path` must be a string literal

> route path must be a string literal. Found in "{id}".

**Level:** Warning

Build-time warning emitted by the file-based routing plugin (`unplugin-vue-router`) while your project builds.

## How to fix it

Use a string literal for the route `path`.
